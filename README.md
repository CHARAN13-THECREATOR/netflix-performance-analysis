# Netflix Customer & Content Performance Analysis  

## 📌 Project Overview  
Explored **customer engagement, subscription patterns, and content trends** for Netflix using **Power BI** and **Excel**.  

## ⚙️ Tools & Skills Used  
- Power BI (Interactive Dashboards, Customer Analytics)  
- Excel (Data Cleaning & Modeling)  
- Data Visualization  

## 📊 Key Insights  
- Audience retention improved by **6%** through recommendations.  
- Targeted genre engagement increased by **15%**.  
- Subscription renewals boosted by **5%**.
